<h1> CurrenSee </h1> <br> <br>
CurrenSee is a personal finance tracker web app built with React, TypeScript, Firebase, and TailwindCSS. It helps users manage their budget, track expenses, and monitor spending habits in real-time. Ideal for anyone who wants to take control of their finances with a clean, responsive interface.<br>
Features<br>
1. User Authentication
2. Sign up with Gmail (email/password)
3. Login, logout, and email verification
4. Budget Management
5. Add budget amounts to track monthly or personal budgets
6. “Budget Left” card shows remaining balance
7. Expense Tracking
8. Record transactions as debit or credit
9. Assign categories to transactions (e.g., Food, Transportation, Entertainment)
10. Real-time updates using Firebase Firestore
11. Transaction History
12. View a detailed transaction history with date, category, and amount
13. Delete transactions and automatically update budgets and totals
14. Analytics & Insights
15. Total Expenses card
16. Top spending category
17. Most recent transaction
18. Responsive Design and fully mobile-friendly with smooth, glassmorphism-style cards and animations.

| Layer              | Technology                     |
| ------------------ | ------------------------------ |
| Frontend           | React + TypeScript             |
| Styling            | TailwindCSS                    |
| Routing            | React Router v7                |
| Backend / Database | Firebase Firestore             |
| Authentication     | Firebase Auth                  |
| Hosting            | Vercel                         |
| Utilities          | React Firebase Hooks, Recharts |
